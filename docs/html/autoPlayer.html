autoPlayer¶

Interface to play games (of Hexathello) and save the results.

Plays games with hexathello.aiPlayers.HexAgent objects.

Returns game results as a JyFrame history class; see hexathello.history.new_literalHistory(), hexathello.history.new.pov_history(). Save these to disk and use them to train AI, most likely hexathello.aiPlayers.KerasHexAgent subclasses.

hexathello.autoPlayer.runHexathello_withAgents(agents: list[AiAgentProtocol], size: int, logging_level: int = 0, rng: Generator | None = None, hexagonGridHelper: HexagonGridHelper | None = None) → JyFrame¶
Parameters:

  • agents (list[ AiAgentProcol ]) – Initialized list of ai agents from aiPlayers

  • size (int) – Length of one side of the hexagon board

  • logging_level (int) – Passed to the hexathello engine for how much detail to log

  • rng (np.random.Generator) – Used to make various decisions both for the game and AI agents

  • hexagonGridHelper (engine.HexagonGridHelper) – Used to help with calculations for both the engine and AI agents

Returns:

The literal game history
